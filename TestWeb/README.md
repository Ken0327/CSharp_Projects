M：產品中心位置（規格中心）
X：群體的中心（平均值）
b = T / 2：	代表規格的一半。
T：規格寬度（規格上限 - 規格下限）
Ck=a/b=(M-X)/(T/2)
CPk = (1 - Ck) * Cp
